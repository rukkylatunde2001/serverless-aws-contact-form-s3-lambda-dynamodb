Data Flow

STEP 1
Static Website
User opens the contact form hosted on S3. HTML/CSS/JS served directly from the bucket.
STEP 2
API Gateway
Form submits a POST request to /contact. CORS enabled.
STEP 3
Lambda Function
Serverless function parses the request and generates a UUID for the submission.
STEP 4
DynamoDB
Submission stored with name, email, message, and ISO timestamp. Zero servers to manage.
